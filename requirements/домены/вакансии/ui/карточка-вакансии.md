# Карточка вакансии

| Название элемента | Формат | Доступность | Обязательность | Input / Output | Описание / Комментарий |
| --- | --- | --- | --- | --- | --- |
| Описание вакансии | Tab | RO (FA если открыта вкладка Кандидаты) | - | - | Выделяется активным цветом, когда открыто описание вакансии (открывается по умолчанию при открытии карточки вакансии) / Если открыта вкладка Кандидаты - по нажатию переключается на вкладку Описание вакансии |
| Кандидаты | Tab | FA | - | - | По нажатию переключается на вкладку Кандидаты |
| Роль | Text | RO | Да | **role:** / name | Отображает информацию из поля "Роль" |
| Подроль | Text | RO | Нет | **subrole:** / name | Отображает информацию из поля "Подроль". / Если значение null, то не отображается |
| Грейд 1 / Грейд 2 | Text | RO | Да | grade | Отображает информацию из поля "Грейд", Грейд 2 отображается в том случае, если было добавлено еще одно значение / Если указано 2 значения, то они отображаются через символ " / " с помощью парсинга |
| Статус | Tag | RO | Да | status | Отображает информацию из поля "Статус" |
| Дата создания | Text | RO | Да | createdAt | Отображает дату создания вакансии в формате ДД.ММ.ГГГГ |
| Датаобращения и время открытия вакансии | Text | RO | Да | dateOfRequest | Отображает информацию из поля "Дата и время открытия вакансии" / Форматы отображаемых данных: / дата - дд.мм.гггг / время - чч:мм |
| Дата и время подготовки резюме | Text | RO | Да |  | Отображает информацию из поля ResumePreparationDateTime. / Форматы отображаемых данных: / дата - дд.мм.гггг / время - чч:мм / Если значение отсутствует, не отображать поле |
| Число сотрудников | Text | RO | Да | quantityOfEmployees | Отображает информацию из поля "Число сотрудников" |
| Локация | Text | RO | Да | location | Отображает информацию из поля "Локация" |
| Метка Метки | Tag | RO | Нет | tags | Отображает информацию из поля"Метка" "Метки" / Если значение null, то не отображается |
| Срок привлечения | Text | RO | Нет | periodOfAttraction | Отображает информацию из поля "Срок привлечения" / Если значение null, то не отображается |
| Ставка для Вендора | Text | RO | Нет | vendorRate | Отображает информацию из поля "Ставка для Вендора" / Если значение null, то не отображается |
| Опыт (лет) | Text | RO | Нет | experience | Отображает информацию из поля "Опыт (лет)" / Если значение null, то не отображается |
| Проект | Text | FA | Да | **project:** / name | Отображает информацию из поля "Проект", по нажатию вызывается метод GET /management/projects/{id}, открывается ЭФ просмотра карточки проекта |
| Команда/Трайб | Text | RO | Нет | tribe | Отображает информацию из поля "Команда/Трайб" / Если значение null, то не отображается |
| Заказчики | Text | RO | Нет | customers | Отображает информацию из поля "Заказчики" / Если значение null, то не отображается |
| Прямой запрос | Text | RO | Да | isDirect | Отображает значение поля "Прямой запрос" / Если значение = true, то отображается "Да"<br>Если значение = false, то отображается "Нет" |
| Требования | Text | RO | Нет | requirements | Отображает информацию из поля "Требования" / Если значение null, то не отображается |
| Кто обрабатывает | Text | RO | Да | **processesBy:** / lastName + firstName | Отображает информацию из поля "Кто обрабатывает" / Если указано несколько значений, то они отображаются через символ " \| " с помощью парсинга |
| Источники поиска | Tag | RO | Да | searchSource | Отображает информацию из поля "Источники поиска" |
| Комментарий | Text | RO | Нет | comment | Отображает информацию из поля "Комментарий" / Если значение null, то не отображается |
| Редактировать | Button | FA | - | - | По нажатию: / вызывает метод GET /management/roles / вызывает метод GET /management/employees / вызывает метод GET /management/projects / открывает ЭФ |
| История вакансии | Button | FA | - | - | По нажатию вызывает метод GET /management/vacancies/{vacancyId}/history, открывает ЭФ просмотра истории изменений по вакансии |
