# Use Case: Создание вакансии

| | |
| --- | --- |
| **Описание** | Создание карточки вакансии. |
| **Предусловия** | Авторизация; список вакансий. |
| **Постусловия** | Вакансия создана. |

## Основной сценарий

1. «Добавить вакансию» → форма.
2. Заполнение → «Создать» → POST `/management/vacancies`.
3. При ошибке валидации — альтернативный сценарий (исправление данных).
