# Use Case: Редактирование вакансии

| | |
| --- | --- |
| **Описание** | Редактирование карточки вакансии. |
| **Предусловия** | Вакансия существует. |
| **Постусловия** | Изменения сохранены. |

## Основной сценарий

1. «Редактировать» → форма → изменения → «Сохранить» → PUT `/management/vacancies/{id}`.
