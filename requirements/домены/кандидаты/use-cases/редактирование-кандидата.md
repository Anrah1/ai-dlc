# Use Case: Редактирование кандидата

| | |
| --- | --- |
| **Описание** | Изменение данных кандидата, импорт Q&A с интервью. |
| **Предусловия** | Кандидат существует. |
| **Постусловия** | Данные обновлены. |

## Основной сценарий

1. «Редактировать» → форма → «Сохранить» → PUT `/management/candidates/{id}`.
2. Импорт файла вопросов-ответов — через pop-up загрузки (см. UI-спеку).
