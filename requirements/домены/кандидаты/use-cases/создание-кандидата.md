# Use Case: Создание кандидата

| | |
| --- | --- |
| **Описание** | Добавление кандидата к вакансии. |
| **Предусловия** | Открыта карточка вакансии или список кандидатов. |
| **Постусловия** | Кандидат создан. |

## Основной сценарий

1. «Добавить кандидата» → форма → заполнение → «Создать».
2. POST `/management/candidates`.
