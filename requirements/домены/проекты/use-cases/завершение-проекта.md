# Use Case: Завершение проекта

| | |
| --- | --- |
| **Описание** | Завершение проекта через боковое меню списка или карточки. |
| **Предусловия** | Проект активен. |
| **Постусловия** | `status = Завершен`. |

## Основной сценарий

1. «Завершить» → диалог подтверждения.
2. «Да» → проверка активных назначений.
3. Нет активных назначений → статус «Завершен», запись в историю.

## Альтернативы

- Уже завершен → «Внимание! Проект уже завершен».
- Есть активные назначения → «Для завершения проекта, завершите все назначения по нему» + ссылки.
- «Отмена» → без изменений.
