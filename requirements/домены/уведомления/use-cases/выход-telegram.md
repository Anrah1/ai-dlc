# Use Case: Выход из Telegram

##### Use case

| **Наименование** |  |
| --- | --- |
| **Функциональная область** | Сервис нотификаций |
| **Описание** | Я, как пользователь, хочу иметь возможность выйти из аккаунта в тг боте |
|  | Пользователь зарегистрирован в системе. / У пользователя есть активный аккаунт в Телеграм. / Пользователь авторизован в системе |
| **Постусловия** | Пользователь вышел из аккаунта в Телеграм боте. |
|  | Пользователь / Сервис нотификации = Телеграм бот / Сервис аутентификации  = Кейклок |

Основной сценарий 1. Выход из аккаунта в боте
**Пользователь **нажимает кнопку "Меню".
**Сервис нотификации** бот возвращает список доступных команд.
**Пользователь** нажимает кнопку "/logout".
**Сервис нотификации **отправляет ChatId пользователя в Сервис аутентификации.
**Сервис аутентификации** ищет пользователя по ChatId
Если пользователь найден переход к п.6
Если пользователь не найден переход к Альтернативному сценарию 2.
**Сервис нотификации** отправляет сообщение с подтверждением выхода "Точно хочешь выйти из аккаунта "username"?".
**Пользователь **Выбирает действие.
Если **Пользователь **хочет выйти Нажимает кнопку "Да" переход к п.6 Основного сценария.
Если **Пользователь** не хочет выходить из системы нажимает кнопку "Нет". Переход к Альтернативному сценарию 1.
**Сервис нотификации** отправляет ChatId пользователя в Сервис аутентификации
**Сервис аутентификации** ищет пользователя по ChatId .
Если пользователь найден переход к п.10 Основного сценария
Если пользователь не найден Переход к п.11 Основного сценария
**Сервис аутентификации** удаляет ChatId.
**Сервис нотификации** отправляет сообщение об успехе операции "Выход выполнен. Для использования функций бота необходимо авторизоваться".
**Пользователь** получает сообщение о том. что он  вышел из аккаунта.
Альтернативный сценарий 1. Пользователь отменил выход из системы
**Сервис нотификации** отправляет сообщение "Выход из аккаунта прерван".
**Пользователь** получает сообщение. Пользователь не вышел из системы
Альтернативный сценарий 2. Пользователь неавторизован в системе
5а. **Сервис аутентификации** не нашел пользователя по ChatId
6. **Сервис аутентификации **возвращает пустой список.
7. **Сервис нотификации** отправляет сообщение "Ты не вошел в аккаунт".
8. **Пользователь** получает сообщение. Пользователь  неавторизован
