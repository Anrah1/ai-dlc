## Введение

Система "Планер" - это инструменты для оптимизации планирования и контроля процесса назначения сотрудников на проекты. Система предназначена для пользования менеджерами проектов, тимлидами и руководством практики.

Система позволяет управлять назначениями, просматривать историю изменений, а также получать необходимые уведомления через telegram-бот.

Данная инструкция пользователя содержит пошаговые инструкции по работе с системой "Планер". В ней подробно описаны основные возможности системы, приведены примеры работы с интерфейсом и даны рекомендации по эффективному использованию инструментов.

Основные возможности системы:

| Процесс | Функционал |
| --- | --- |
| Управление назначениями | Заведение сотрудника, проекта, назначения / Внесение изменений о сотруднике, проекте, назначении / Изменения статуса сотрудника (работает, уволен и т.д.), проекта (активный, завершен), назначения (активно, завершено и т.д.) / Экспорт данных в формате Excel. |
| История изменений | Система автоматически отслеживает и хранит историю изменений. / Просмотр истории изменений по конкретному сотруднику, проекту, назначению. |
| Уведомления | С помощью telegram-бота: / Получение уведомлений о дате подключения сотрудника на проект. / Получение уведомлений о дате отключения сотрудника на проекте. / Получение уведомлений о дате оплаты сотрудника на проекте. |

#### Просмотр истории изменений

Описание

Для того, чтобы открыть историю изменений сотрудника/проекта/назначения необходимо открыть карточку сотрудника/проекта/назначения.

В карточке слева снизу расположена кнопка "История изменений" (), при нажатии на кнопку открывается история изменений.

![Иллюстрация](images/instrukciya-3.png)

Figure 2 Рис. Просмотр истории изменений

В открывшемся окне отображается следующее ():

![Иллюстрация](images/instrukciya-4.png)

Figure 3 Рис. История изменений

Строка поиска (1) - используется для сквозного поиска по заданному значению из списка.

Информация по истории изменений (2):

Дата изменения - дата и время, когда происходили изменения.

Измененное поле - какое поле изменяли в карточке.

Предыдущее значение - значение изменяемого поля, которое было установлено до изменения; если в поле в конце указано "...", то при наведении отображаются все значения.

Новое значение - новое значение изменяемого поля; если в поле в конце указано "...", то при наведении отображаются все значения.

Изменил(а) - кто вносил изменения (Фамилия и имя сотрудника).

Счетчик строк и навигация (3) - отображает общее количество строк в списке, позволяет перемещаться по страницам списка.

Сортировка (4) - используется для сортировки значений в столбце.

Фильтрация (5) - используется для фильтрации значений в столбце.

По кнопке (6) можно закрыть историю изменений.

##### Создание внутреннего сотрудника

Описание

В открывшемся окне (Рис. Создание внутреннего сотрудника) необходимо заполнить следующие поля:

![Иллюстрация](images/instrukciya-8.png)

Figure 7 Рис. Создание внутреннего сотрудника

"Тип сотрудника"** (1)** - выпадающий список с выбором типа сотрудника, позволяет переключить карточку создания с внутреннего сотрудника на внешнего и обратно. Значение "Внутренний" выставлено по умолчанию.

"Фамилия"** (2)** - обязательное для заполнения. Поле для свободного ввода фамилии сотрудника.

"Имя"** (3)** - обязательное для заполнения. Поле для свободного ввода имени сотрудника.

"Отчество (при наличии)"** (4)** - поле для свободного ввода отчества сотрудника. Допускается оставлять поле пустым.

"Роль" **(5)** - обязательное для заполнения. Выпадающий список с выбором роли сотрудника.

"Подроль" **(6)** - выпадающий список с выбором соответствующей подроли для выбранной роли сотрудника. Допускается оставлять поле пустым.

"Грейд" **(7)** - обязательное для заполнения. Выпадающий список с выбором грейда сотрудника.

"Руководитель" **(8)** - обязательное для заполнения (кроме сотрудников с ролью Тимлид или Менеджер, для них допускается оставлять поле пустым). В списке показаны и доступны для выбора сотрудники с ролью Тимлид или Менеджер.

"Дата рождения" **(9)** - обязательное для заполнения. Поле с календарем, позволяет выбрать день и месяц рождения сотрудника. Год рождения не указывается.

"Комментарий" **(10)** - поле для свободного ввода дополнительной информации о сотруднике. Допускается оставлять поле пустым.

После заполнения всех нужных и обязательных полей необходимо нажать на кнопку **"Создать"** (№12 Рис. Создание внутреннего сотрудника) для создания новой карточки сотрудника.

Для отмены создания можно нажать на кнопку **"Отменить"** (№11 Рис. Создание внутреннего сотрудника), нажать на **крестик** (№13 Рис. Создание внутреннего сотрудника) или нажать на клавиатуре на клавишу **esc**.

### Работа с проектами

#### Создание проекта

Описание

Для заведения проекта в системе пользователю необходимо перейти на вкладку «Проекты» в боковом меню (п. на Рис.  Список проектов). На экране отображается список всех проектов (п.1  на Рис.  Список проектов)

![Иллюстрация](images/instrukciya-14.png)

Figure 13 Рис. Список проектов

Для того, чтобы создать новый проект необходимо нажать кнопку "Создать проект"   (п.2 на Рис.  Список проектов). На открывшемся окне (Рис.  Создание проекта) заполнить следующие поля:

![Иллюстрация](images/instrukciya-15.png)

Figure 14 Рис. Создание проекта

Название - обязательное поле, поле свободного ввода, предназначено для ввода названия проекта. Должно быть уникальным и отличаться от предыдущих. (п.1 Рис.  Создание проекта)

Менеджер -  обязательное поле, при нажатии на поле открывается выпадающий список, из которого необходимо выбрать сотрудника с ролью менеджер. (п.2 Рис.  Создание проекта)

Код проекта - поле свободного ввода, можно ввести несколько ЧК проекта. Для добавления нового значения в поле необходимо ввести название ЧК и нажать Enter. (п.3 Рис.  Создание проекта)

Тип проекта - по умолчанию: внутренний. (п.4 Рис.  Создание проекта)

Комментарий - поле свободного ввода, предназначено для ввода дополнительной информации (п.5 Рис.  Создание проекта)

По кнопке «Создать» (п.7 на Рис.  Создание проекта) Создается новая запись проекта в списке. (п.1  на Рис.  Список проектов)

По кнопке «Отменить» (п.6 на Рис.  Создание проекта) и кнопке (8) отменяется создание проекта и форма закрывается без сохранения.

#### Завершение проекта

"Сотрудник" (1) - обязательное поле, при нажатии на поле открывается выпадающий список, из которого необходимо выбрать сотрудника, который назначается на проект. Доступен поиск по сотрудникам, для этого необходимо ввести первые буквы фамилии/имени. В списке отображаются сотрудники со статусом "Работает".

"Роль" (2) - обязательное поле, после выбора сотрудника из его карточки автоматически заполняется данное поле. Имеется возможность поменять роль, при нажатии на поле открывается выпадающий список со всеми ролями.

"Подроль" (3) - данное поле недоступно, если выбрана роль, у которой нет подролей; после выбора сотрудника из его карточки автоматически заполняется данное поле. Имеется возможность поменять подроль, при нажатии на поле открывается выпадающий список с соответствующими для выбранной роли подролями.

"Грейд" (4) - обязательное поле, после выбора сотрудника из его карточки автоматически заполняется данное поле. Имеется возможность поменять грейд, при нажатии на поле открывается выпадающий список грейдов.

"Проект" (5) - обязательное поле, при нажатии на поле открывается выпадающий список с названиями проектов, имеющих статус "Активный". Доступен поиск по проектам, для этого необходимо ввести первые буквы названия проекта.

"Заказчики" (6) - обязательное поле, поле свободного ввода.

"Ответственный за обратную связь" (7) - поле свободного ввода, предназначено для ввода ответственного за обратную связь сотрудника, назначающегося на проект.

"Условия" (8) - обязательное поле, при нажатии на поле открывается выпадающий список с условиями подключения на проект.

"Даты оплаты" (9) - обязательное поле, при нажатии на поле открывается календарь: нажав на определенное число, оно добавляется в поле. Можно убирать и менять даты в календаре, либо нажать на крестик рядом с датой, чтобы убрать выбранную дату, либо навести на значок календаря в поле: отобразится крестик, нажав на него, все даты в поле очистятся. Максимум можно выбрать 20 дат.

"Дата подключения - дата отключения" (10) - обязательное поле, при нажатии на поле открывается календарь: в левой части календаря выбирается дата подключения, в правой - дата отключения. Можно менять даты в календаре, либо навести на значок календаря в поле: отобразится крестик, нажав на него, все даты в поле очистятся.

"Ставка без НДС., руб. в час/Проценты" (11) - доступен ввод цифрами, при выборе "Условия" (8) отображается необходимое поле: Если выбраны условия "Профитшеринг"/"Ставка", то в данном поле происходит ввод ставки без НДС., руб . в час; если выбрано условие "ЧК", то происходит ввод процентов.

"Условия по ЧК" (12) - ввод состоит из трёх обязательных полей: ЧК откуда (свободный ввод), ЧК куда (свободный ввод), тип (Доход/Расход).
Любую строку можно удалить, для этого необходимо нажать на кнопку (13).
Можно добавить строки, для этого необходимо нажать на кнопку (14).

"Комментарий" (15) - поле свободного ввода.

По кнопке "Создать" (16) создается новая запись назначения.
По кнопке "Отменить" (17) и кнопке (18) отменяется создание назначения и форма закрывается.

#### Создание вакансии

Описание

Для создания новой вакансии в системе пользователю необходимо перейти на вкладку «Вакансии» в боковом меню (п.1 Рис.  Список вакансий). Затем на вкладку "Список вакансий"(п.2  Рис.  Список вакансий). На экране отображается список всех вакансий (п.3  Рис.  Список вакансий).

Необходимо нажать кнопку "**Добавить вакансию**"  (п.4  Рис.  Список вакансий).

Figure 32 Рис. Список вакансий

![Иллюстрация](images/instrukciya-34.png)

Figure 33 Создание вакансии

Дата обращения - обязательное поле, предзаполнено текущей датой. Для изменения значения необходимо нажать на поле и выбрать нужную дату.

Число сотрудников - обязательное поле, предзаполнено значением 1.  Для изменения значения можно воспользоваться стрелками вверх и вниз в правой части поля или вручную ввести значение.

Срок привлечения - поле свободного ввода, предназначено для ввода срока на который планируется задействовать кандидата на вакансии.

Роль - обязательное поле, при нажатии на поле открывается выпадающий список, из которого необходимо выбрать роль на которую открыта вакансия.

Подроль -  при нажатии на поле опционально открывается выпадающий список, из которого необходимо выбрать подроль на которую открыта вакансия.

Грейд - обязательное поле, при нажатии на поле открывается выпадающий список, из которого необходимо выбрать один или два грейда, на которые  ведется поиск.

Проект - обязательное поле,  при нажатии на поле открывается выпадающий список с названиями проектов. Доступен поиск по проектам, для этого необходимо ввести первые буквы названия проекта.

Заказчики - поле свободного ввода, предназначено для ввода ФИО и контактной информации о заказчиках.

Кто обрабатывает - обязательное поле,  при нажатии на поле открывается выпадающий список, из которого необходимо выбрать от одного до трех сотрудников, которые работают с данной вакансией.

Метка - поле свободного ввода. Для добавления нового значения в поле необходимо ввести название метки и нажать Enter. Максимальное количество меток - 5.

Требования - поле свободного ввода, предназначено для ввода перечня требований по вакансии.

Комментарий -  поле свободного ввода, предназначено для ввода дополнительной информации.

ОС по интервью (внутр.) - поле свободного ввода, предназначено для ввода обратной связи по внутреннему интервью.

Дата интервью (внеш.) - при нажатии на поле открывается календарь: нажав на определенное число, оно добавляется в поле. Максимум можно выбрать 3 даты.

ОС по интервью (внеш.) - поле свободного ввода, предназначено для ввода обратной связи по внешнему интервью.

Комментарий - поле свободного ввода, предназначено для ввода дополнительной информации.

По кнопке «**Загрузить резюме**» (10) выполняется загрузка в систему резюме кандидата:

Необходимо выбрать файл с резюме из проводника.** Файл должен соответствовать следующим требованиям:**

Максимальный размер файла - 10 МБ;

Максимальная длина названия файла - 30 символов;

Допустимые форматы для загрузки: .pdf, .doc, .docx.

Результат загрузки файла отобразится всплывающим уведомлением. При успешной загрузке название файла отобразится ниже кнопки.

Для того, **чтобы удалить файл** необходимо навести курсор на название файла и нажать на кнопку "корзина".

| Система хранит только последний загруженный файл, т.е. при повторной загрузке происходит замена загруженного файла. |

По кнопке «**Создать**» (12) создается новая запись кандидата и отображается на списке кандидатов вакансии.

По кнопке «**Отменить**» (11) и кнопке (13) отменяется создание вакансии и форма закрывается без сохранения.

#### Импорт файла вопросов-ответов с интервью

Описание

Для каждого кандидата имеется возможность выгрузить файл Excel с вопросами-ответами с интервью. Для этого необходимо на карточке кандидата нажать на кнопку «**Открыть вопросы****»** ().

После нажатия открывается форма с вопросами и ответами на интервью:

По нажатию кнопки «**Загрузить вопросы» **(1)** **открывается форма для загрузки файла ().
Вопросы и ответы отображаются на форме (2) с возможностью скрыть ответ на вопрос, нажав на кнопку (3).
Счетчик строк (4) отображает количество вопросов. Навигация (5) позволяет перемещаться по вопросам.

По кнопке (6) форма просмотра вопросов-ответов с интервью закрывается.

По кнопке «**Перетащите или выберите файл****» **(1)** **открывается проводник для выбора файла.

**Требования к файлу Excel:**

Максимальный размер файла - 10 МБ;

Максимальная длина названия файла - 30 символов;

Допустимые форматы для загрузки: .xlsx, .xls;

Файл содержит два столбца: 1 столбец - вопросы, 2 столбец - ответы. В файле должны отсутствовать заголовки (первая строка - первый вопрос и ответ).

После загрузки через проводник название файла отображается на форме (2) с возможностью удалить файл.

По кнопке «**Загрузить****» **(3)** **выполняется импорт файла, отображается уведомление о загрузке и форма закрывается.

#### Подключение бота

Описание

Откройте бот в Telegram:

Пройдя по ;

Найдя его с помощью глобального поиска - **@liga_planer_bot**;

После приветственного сообщения нажмите кнопку "Начать".

Прочитав авторизационное сообщение, нажмите кнопку "Войти в систему".

Дата отключения сотрудника от проекта.

Дата оплаты.
