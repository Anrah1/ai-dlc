#### Описание

Сервис нотификации — отвечает за отправку уведомлений пользователям через Telegram бота.
Микросервис взаимодействует с другими частями системы, а именно , обрабатывает запросы на отправку уведомлений и обеспечивает надежную и безопасную доставку сообщений.
Основная функция микросервиса - получение запросов на уведомление. Сервис нотификации слушает kafka notification-topic от , который инициирует отправку уведомлений.
На основе полученных данных формируется тело сообщения, которое будет отправлено пользователю через Telegram бота. Учитываются шаблоны сообщений для различных типов уведомлений.
Микросервис использует Telegram Bot API для отправки сообщений пользователям.
Используется брокер сообщений - Kafka для асинхронной обработки отправки уведомлений, чтобы избежать перегрузки и обеспечить масштабируемость. Микросервис ведет журнал всех отправленных уведомлений, включая время отправки, статус и текст сообщения.

#### 2) Отправка уведомлений через телеграм бот

Сервис нотификации отправляет данные пользователя в Сервис аутентификации и получает соответствующий Chat ID. По Chat ID. сервис отправляет сформированное сообщение в нужный тг чат пользователю.
