# Импорт резюме

**POST** `/management/files`

Описание
**Описание метода:**
Метод предназначен для импорта файла резюме (без парсинга)
Метод: POST
Эндпоинт: https://management/files
**Строка**** запроса****:** POST https://management/files
**Логика работы метода:**
Принять файл, направленный через HTTP-запрос (multipart/form-data)
Сохранить файл в хранилище, присвоить UUID
Записать UUID файла в таблицу БД planer, соответствующую сущности, к которой прикрепляется файл (candidates.file_uuid
- employees.file_uuid)
Вернуть ответ на запрос с UUID

| **Название параметра** | **Тип данных** | **Обязательность** | **Описание** | **Мэппинг на БД** |
| --- | --- | --- | --- | --- |
| file | multipart/form-data | + | Файл резюме, который нужно загрузить в систему. | Хранится в файловом хранилище. |

**Ответ:**

| **Название параметра** | **Тип данных** | **Обязательность** | **Описание** | **Мэппинг на БД** |
| --- | --- | --- | --- | --- |
| fileUuid | uuid | + | Универсальный уникальный идентификатор (uuid) файла  резюме в хранилище | planer.candidates.file_uuid / planer.employees.file_uuid |

200 ОК

| { / "fileUuid": "a0eebc99-9c0b-4ef8-bb6d-6bb9bd380a11" / } |
| --- |

204 No Content

| { / "fileUuid": **null** / } |
| --- |

**Ошибки:**
400 Bad Request: Некорректные данные запроса (отсутствует файл)

| { / "status": "error", / "message": "No file part" / } |
| --- |
