# Получение списка ролей с подролями

**GET** `/management/roles`

Получение списка ролей с подролями
**Описание метода:**
Метод предназначен для получения списка всех ролей и соответствующих им подролей
Метод: GET
Эндпоинт: /management/roles
**Строка запроса: **GET https:// ... /management/roles
**Тело запроса:**
Отсутствует
**Примеры ответов:**
Успешный ответ:
200 OK
