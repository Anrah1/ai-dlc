# Конвертация резюме

**POST** `/management/candidates/{candidateId}/resume/convert`

Описание
**Описание метода:**
Метод предназначен для конертации файла резюме в лиговский шаблон
Метод: POST
Эндпоинт:  /management/candidates/{candidateId}/resume/convert
**Строка**** запроса****:** POST /management/candidates/{candidateId}/resume/convert
**Логика работы метода:**
Инициирует синхронный процесс:
загрузку исходного резюме кандидата по resumeUuid,
извлечение текста,
отправку текста в нейроAPI с указанным промптом для получения текста резюме в типе markdown,
конвертацию из типа markdown в DOCX,
сохранение DOCX в файловое хранилище,
обновление поля convertedResumeUuid у кандидата,
возврат результата клиенту.

| **Название параметра** | **Тип данных** | **Обязательность** | **Описание** | **Мэппинг на БД** |
| --- | --- | --- | --- | --- |
| candidateId | path-param | + | UUID кандидата, для которого выполняется конвертация | planer.candidates.id |

**Запрос:**

| **Название параметра** | **Тип данных** | **Обязательность** | **Описание** | **Мэппинг на БД** |
| --- | --- | --- | --- | --- |
| resumeUuid | uuid | + | UUID резюме для конвертации | planer.candidates.file_uuid |
| prompt | string | - | Версия промпта |  |

**Ответ:**

| **Название параметра** | **Тип данных** | **Обязательность** | **Описание** | **Мэппинг на БД** |

| convertedResumeUuid | uuid | + | Универсальный уникальный идентификатор (uuid) файла сконвертированного резюме в хранилище | planer.candidates.converted_resume_uuid |
