# API: кандидаты

Методы группы «кандидаты».

| Метод | Эндпоинт | Описание |
| --- | --- | --- |
| GET | `/management/candidates` | [Получение списка кандидатов](get-получение-списка-кандидатов.md) |
| GET | `/management/candidates/{candidateId}` | [Получение информации о конкретном кандидате](get-получение-информации-о-конкретном-кандидате.md) |
| GET | `/management/candidates/{candidateId}/history` | [Получение истории изменений кандидата](get-получение-истории-изменений-кандидата.md) |
| GET | `/management/candidates/{candidateId}/interview-results` | [Получение информации с вопросами-ответами кандидата на интервью](get-получение-информации-с-вопросами-ответами-кандидата-на-интер.md) |
| PUT | `/management/candidates/{candidateId}/interview-results` | [Импорт файла с вопросами-ответами кандидата на интервью](put-импорт-файла-с-вопросами-ответами-кандидата-на-интервью.md) |
| POST | `/management/candidates/{candidateId}/resume/convert` | [Конвертация резюме](post-конвертация-резюме.md) |
