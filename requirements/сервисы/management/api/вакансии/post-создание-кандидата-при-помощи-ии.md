# Создание кандидата при помощи ИИ

**POST** `/management/vacancies/ai/{vacancyId}`

Описание
**Описание метода:**
Метод предназначен для создания нового кандидата через ИИ
Валидации:
Метод: POST
Эндпоинт: https://management/vacancies/ai/{vacancyId}
**Строка** **запроса****:** POST https://management/vacancies/ai/{id}
Логика метода:
Клиент отправляет запрос на создание кандидата с ИИ POST }.
Сервер создаёт запись кандидата в БД (если валидация пройдена), генерирует jobId и запускает фоновый процесс.
Сервер немедленно возвращает ответ с candidateId, jobId и статусом PROCESSING.
Фоновая задача:
Если передан resumeUuid, загружает файл резюме из хранилища и с помощью скрипта-парсера извлекает текст.
Получает актуальные данные вакансии по vacancyId (делает снэпшот описания вакансии).
Формирует промт и отправляет запрос в ИИ.
Получает ответ, парсит scorePercent и summaryText.
Сохраняет запись в таблицу candidate_ai_match, заполняя все поля, включая jobId.
Клиент может периодически опрашивать GET /management/vacancies/ai/{jobId} для получения статуса и результата.
